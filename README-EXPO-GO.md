# Sainat AI - Mobil Uygulama

## Expo Go ile Çalıştırma

### 1. Gereksinimler
- Expo Go uygulaması ( telefonunuzda yüklü olmalı)
- Node.js (bilgisayarınızda)

### 2. Başlatma Adımları

```bash
# 1. Terminali açın ve proje klasörüne gidin
cd C:\Users\acer\CascadeProjects\sainat.art\sainat-mobile

# 2. Bağımlılıkları yükleyin
npm install

# 3. Expo development server'ı başlatın
npx expo start
```

### 3. Uygulamaya Erişim Seçenekeleri

#### 📱 **Mobil (Expo Go)**
1. Terminalde QR kod görünecek
2. Expo Go uygulamasını açın
3. "Scan QR Code" butonuna basın
4. Telefon kamerası ile QR kodu okutun
5. Uygulama otomatik olarak açılır

#### 🌐 **Tunnel Mode (Her Yerden Erişim)**
1. Terminalde `npx expo start --tunnel` çalıştırın
2. Terminalde görünen **exp://XXX.tunnelto.dev** URL'ini kopyalayın
3. Expo Go'da manuel olarak yapıştırın
4. Her yerden erişim sağlar

#### 💻 **Development Server**
- **Local:** exp://localhost:8081
- **Tunnel:** exp://[RASTGELE].tunnelto.dev (terminalde gösterilir)
- **QR Code:** Mobil cihazdan hızlı bağlantı
